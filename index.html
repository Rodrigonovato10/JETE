Posições rotativas

São as vagas disputadas no revezamento. Se a equipe ganhar/perder uma posição (ex: abrir uma 2ª "ESTEIRA / MEIO"), adicione ou remova aqui.

Folgas fixas da semana

Marque quem folga em cada dia. Domingo não é gerado automaticamente pelo revezamento — marque manualmente no campo abaixo.

Folgas de Domingo (manual)

Domingo não tem revezamento automático. Marque aqui quem folga neste domingo — pode marcar vários de uma vez.

Observações (restrições de posição)

Use para casos como "Carlos hoje não vai para B1". A pessoa segue disponível para revezar nas outras posições, só fica bloqueada para essa específica.

Atribuição manual (forçar pessoa numa posição)

Define que uma pessoa específica fica numa posição específica, sobrepondo o revezamento automático. Ela é retirada do sorteio das outras posições naquele dia.
